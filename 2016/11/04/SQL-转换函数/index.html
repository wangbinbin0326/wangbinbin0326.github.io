4. 使用转换函数和条件表达式

隐式数据类型转换, Oracle Server 可以自动执行以下转换:

  • VARCHAR2 或CHAR —> NUMBER
  • VARCHAR2 或CHAR —> DATE
  • NUMBER —> VARCHAR2 或CHAR
  • DATE —> VARCHAR2 或CHAR

注： number 与date 不能隐式转换
